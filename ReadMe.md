# 简介
这是一个智能相册应用，可以方便地管理和浏览本地照片，集成了 ComfyUI 接口服务，让使用者能够快速调用 AI 生成或处理图像的功能。

# 使用说明
1. 进入主界面后，在任意区域输入手势“Z”即可解锁相册界面。如果想要取消此界面，可在主界面的输入框中输入unlock并发送。长按相册界面的标题可以恢复此界面。
2. 首次进入相册界面，请在右上角菜单中进行地址设置，输入后端地址，格式为“http://192.168.1.1:8000/”，不要遗漏“/”，输入完后勾选该地址。
3. 相册界面右上角菜单中可以批量添加图片、查询当前任务、刷新当前页面。
4. 菜单中的刷新页面除刷新目录之外，还会重新进行当前文件夹下图片资源的数据库扫描。如果没有手动修改文件，请下拉相册界面刷新。
5. 点击底部的tab可以切换文件夹，双击tab可以快速返回顶部。
6. 拖动右侧的滑块可以快速定位图片。
7. 在相册界面长按图片进入多选模式，同时弹出操作栏。功能请自行探索。
8. 在大图界面点击图片同样可以弹出操作栏。
9. 在大图界面左右滑动可以切换图片，双击缩放图片，下滑退出大图模式。
10. 大图模式采用预加载模式，首先加载缩略图，一段时间内图片模糊是正常的。
